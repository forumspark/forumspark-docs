# Directory Listing

The ForumSpark Directory is a public listing of community forums. Listing your board in the directory helps new members discover your community.

You can manage your directory listing from the **ForumSpark** section in the Admin Control Panel.

[[toc]]

## Opting In

By default, your board is not listed in the directory. To opt in:

1. Go to **ForumSpark > Directory Listing** in the Admin CP.
2. Toggle **Listed in Directory** to on.
3. Select a **Category** that best describes your community.
4. Write a short **Description** (max 500 characters) to help visitors understand what your forum is about.
5. Save your changes.

## Screenshots

When your board is listed in the directory, a screenshot of your forum homepage is automatically captured and displayed alongside your listing. This gives potential members a preview of your community.

You can request a new screenshot at any time from the directory settings page. This is useful if you've recently changed your theme or layout and want the listing to reflect the update.

## Notes

- Only boards with an active directory listing will appear in the public directory.
- The description field supports plain text only (no BBCode or HTML).
- Changes to your listing are synced to the directory when you save.
- You can opt out of the directory at any time by toggling the listing off.
