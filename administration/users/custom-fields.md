# Custom Profile Fields

Custom profile fields let you collect additional information from your members beyond the standard profile. Fields can be displayed alongside user posts in the mini profile area. You can manage them under the **User Management** navigation group in the Admin CP.

## Creating a field

Click **Create Field** to add a new custom profile field. Each field has the following properties:

- **Handle** — a unique internal identifier used to reference the field (letters, numbers, dashes and underscores only).
- **Label** — the display name shown to users when editing their profile.
- **Sort Order** — controls the order in which fields appear on the profile and in the mini profile.
- **Show in Mini Profile** — when enabled, the field value is displayed next to the user's posts.

## Field types

Currently, custom profile fields support a **single-line text** input. Users can enter up to 255 characters per field.

## Mini profile display

Fields with **Show in Mini Profile** enabled are shown alongside every post a user makes, in the sidebar area next to their avatar and username. This is useful for fields like location, occupation or a favourite quote.

Fields appear in the mini profile in ascending order based on their **Sort Order** value.

## How users fill in fields

Members can fill in their custom profile fields from the **Edit Profile** page. Each field appears with its label and a text input. Changes are saved alongside the rest of their profile information.

## Editing and deleting fields

Click on any field in the list to update its properties. Deleting a field removes it from all user profiles — any responses that members have entered for that field are also removed.
